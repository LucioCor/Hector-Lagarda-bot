Hector-Lagarda bot:
Discord bot capable of reproducing audio files, quotes and gifs of my friend Hector.
This bot was made with basic knowledge of python, following the guides of
IT Connected youtube channel: https://www.youtube.com/channel/UCyte7e0lLMQW3s4Oox-7rQQ,
and examples from https://github.com/Rapptz/discord.py

Built with:
Python: https://www.python.org/
Discord: https://discordapp.com/
Recommeded code editor: Visual Studio Code: https://code.visualstudio.com/
FFmpeg: https://ffmpeg.zeranoe.com/builds/
youtube-dl: https://rg3.github.io/youtube-dl/
pip install from discord: pip install discord.py[voice]

Commands:
Type !help for the following list of commands:

!help
